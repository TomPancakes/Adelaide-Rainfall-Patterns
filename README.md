
# Adelaide Rainfall Analysis :)

## Overview: 
Analysis of rainfall patterns in Adelaide, using historical data from BOM spanning 1839-2026. (https://www.bom.gov.au/). This project was written part curiosity, part python data analysis practice. 
 
## Key Findings: 
- Total annual rainfall seems to have remained reasonably stable since beginning recording. 
- Winter months show a slightly more pronounced downward trend, but not singnifcant
- Rain variability has slightly increased post 1970 vs prior. (SD: 107-117) 

## Tools used
- Pandas (Data Processing)
- Matploblib (Graphing)
- Numpy (trend lines)

## Data Sources
- BOM weather stations West Terrace & Kent Town were used for data. 
- West Terrace: All dates bar 1970-2019. 
- Kent Town: 1970-2019
